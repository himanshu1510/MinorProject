## How the setup the project in your system
### 1. Setting up the local server
#### Go to the NodeJs rest-api folder and install the the required libraries by running the command **npm install**
#### Starting you local server run command **node server.js**
![image](https://user-images.githubusercontent.com/72853645/138012160-866bb80b-4bfb-4308-b810-ff9ebcd88e85.png)

### 2. Setting you ngrok to locally running web service.
#### Download ngrok for you system 
#### run the command **ngrok http 3000** to temporarily 
![image](https://user-images.githubusercontent.com/72853645/138012234-ed9ee70d-31be-4b7e-80ad-284cf723f5f8.png)
![image](https://user-images.githubusercontent.com/72853645/138012272-45f13fa4-7ccd-4f48-b17d-52411f0445f3.png)

### 3. Setting you project App
#### Go the the project folder and run the command **npm install** to install all the dependencies and required packages.
#### run the command to start your app on emulator or your device **npm run android**
